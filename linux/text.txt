CTRL + ALT + T - КОМАНДА ОТКРЫТИЕ ТЕРМИНАЛА 
~ - СИМВОЛ ДОМАШНЕЙ ДЕРИКТОРИЙ(ПАПКИ)
/ - СИМВОЛ КОРНЕВОЙ ДЕРИКТОРИЙ(ПАПКИ)
.. - ОБОЗНАЧАЕТ ПРЕДЫДУЩИЕ ДЕРИКТОРИИ 
. - ТЕКУЩАЯ ДЕРИКТОРИЯ 

1. LS(LIST) - > Выводит список всех директорий и файлов в текущей дериктории(рабочая директория)
ls -a -> Команда выводит список как и скрытых и так и обычных дерикторий 
ls -l -> Выводит список дерикторий и файлов с подробной информацией
ls -la -> Выводит все файлы и дериктории 
2.CD (change directory) - > Меняет текующую дерикторию на указаный 
cd <название дериктории>
3. mkdir (make direkrory) - > Создает указанную дерикторию mkdir <директория>
4.touch - > Создает указанный вайл
touch <name_of_file>.<extension(расширение(txt, docs))>
5.pwd (print working directory) - > выводит путь до текущей дериктории 
6. rmdir (remove directory) - > Удаляет указанную директорию 
7. CTRL + L - Очистка терминала 
8. RM - > Удаляет файл 
rm<name_of_file>
rm -rf -> принудительное удаления файла или дериктории 
9. mv(move) -> Переименование файла и перемещение файла
mv <old_name> <new_name> - Переименование
mv <file> <на существующий файл> - перемещение 
10. cat - > Выводит содержимое указанного файла 
11. nano -> открывает графический интерфейс в терминале для редактирование файлов
*если нет указанного файла, то создасться новый файл с таким именем*
nano <name_of_file> 

12. echo - >  Команда для бытрой записи данных в файле через терминал 
ech 'текст для записи' > название файла 

13.cp (copy) -> команда для копирования файла в нужную дерикторию 
cp <file_to_copy> <path>
